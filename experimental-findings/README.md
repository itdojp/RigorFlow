## 📝 事例追加ガイドライン

新しい実験結果を追加する際は、以下の形式に従ってください：

1. **ディレクトリ名**: `YYYY-MM-プロジェクト略称/`
2. **必須ファイル**:
   - `README.md` - 事例の概要
   - `lessons-learned.md` - 得られた知見
   - その他関連ドキュメント

3. **推奨構成**:
   ```
   YYYY-MM-project-name/
   ├── README.md           # 事例概要
   ├── lessons-learned.md  # 知見まとめ
   ├── metrics/           # 測定結果
   ├── issues/            # 課題と改善
   └── artifacts/         # 成果物
   ```
